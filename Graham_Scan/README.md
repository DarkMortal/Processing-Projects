# Convex Hull using Graham Scan

## [Video I followed](https://youtu.be/B2AJoQSZf4M)

## Controls
- Click anywhere on canvas to add points
- <kbd>S</kbd> to draw the convex hull
- <kbd>R</kbd> to reset the canvas
