## Controls
- Use to arrow keys to move the shape
- Press <kbd>L</kbd> key for rotating the shape clock-wise
- Press <kbd>R</kbd> key for rotating the shape anticlock-wise
- Shape selection:
  - <kbd>S</kbd> : Square
  - <kbd>T</kbd> : Triangle
  - <kbd>Q</kbd> : Quadrilateral
